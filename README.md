# KlasifikasiBatik
Klasifikasi  batik   adalah  sebuah  Program  yang  dapat mendeteksi  dan  mengklasifikasikan  pola gambar batik dengan menggunakan metode Deep learning yaitu Convolutional Neural Network (CNN). Menggunakan Bahasa pemrograman Python dan library dari Keras.

Dataset yang digunakan bersumber dari https://www.kaggle.com/datasets/dionisiusdh/indonesian-batik-motifs?resource=download
